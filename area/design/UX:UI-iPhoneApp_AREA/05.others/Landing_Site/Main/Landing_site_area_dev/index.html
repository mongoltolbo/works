街の今を、知ろう

もし、空を飛べなかったら、その地上絵はただの土だったろう。

もし、AREAがなかったら、その街はただのコンクリートだっただろう。

街の今を、あなたのこれからに、AREA。


ツイート
地図で好きな場所をタッチ。知ってる所も知らない所も、AREAにお任せ。もちろん、今いる場所の周辺情報もわかります。
まるでリアルタイムに発行されるフリーペーパー。AREAなら、「街の今」が手に取るようにわかります。
AREAで気になったお店の様子をチェック。行きたいなと思ったら、実際に行ってみましょう。お店もあなたもハッピーに。
